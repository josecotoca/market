### Installation

1. Clone the repo [https://github.com/josecotoca/my-store](git@github.com:josecotoca/my-store.git)
  ```sh
   git clone git@github.com:josecotoca/my-store.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
